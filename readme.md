# To Install in Ubuntu:
```java
sudo apt update
sudo apt install screenfetch -y

```
Tree
```java
sudo apt update
sudo apt install tree -y

```


# To start screenfetch on startup:
`nano ~/.bashrc` <br/>
then add at the very end <br/>
```java
# Loading Screenfetch
screenfetch
```
close save, restart
